Awesome route graph with MojoX::Routes::AsGraph

published on May 27, 2009
We are testing out Sebastian's Mojolicious framework for use in a Nordaaker project. Mojolicious uses an routes-based dispatcher, which is nice, but sometimes it's hard to keep track of how everything is routed. Luckily Melo already has an excellent solution for this: Just check out MojoX::Routes::AsGraph , which can generate these lovely graphs for you. It can even generate Ascii graphs, if you're a hard-core console user (Thanks to vti for telling me about that, as well as showing me this module in the first place :). Read More...
