Creating a standalone Damon with Moose and friends

published on January 30, 2009
I recently needed to create a persistent daemon to do some maintenance tasks for iusethis. Turns out that building these with the postmodern object system Moose is really easy. For those of you not in the know, Moose has the concept of 'Roles', stolen from the Perl 6 project. Roles work kind of like a mix-in that you can add to your class to make it super powerful :) (For the extra curious, you can read more details about Roles in Perl6 here). Read More...
