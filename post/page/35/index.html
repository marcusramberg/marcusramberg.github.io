published on May 27, 2009
We are testing out Sebastian's Mojolicious framework for use in a Nordaaker project. Mojolicious uses an routes-based dispatcher, which is nice, but sometimes it's hard to keep track of how everything is routed. Luckily Melo already has an excellent solution for this: Just check out MojoX::Routes::AsGraph , which can generate these lovely graphs for you.
